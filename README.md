# Şifâhen GitHub Pages Site

Bu klasör, GitHub Pages üzerinde yayınlanacak Şifâhen tanıtım sitesi dosyalarını içerir.

## Yayınlama

Bu klasördeki dosyaları `sifahen-privacy` reposunun kök dizinine yükleyin:

- `index.html`
- `privacy-policy.html`
- `assets/`

GitHub Pages ayarı:

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/ (root)`

## URL'ler

- Tanıtım sitesi: `https://esraaksoy29.github.io/sifahen-privacy/`
- Gizlilik politikası: `https://esraaksoy29.github.io/sifahen-privacy/privacy-policy.html`

Google Play gizlilik politikası alanında `privacy-policy.html` URL'si kullanılmalıdır.
